브랜치 전환하기
me@work MINGW64 ~

git switch [브랜치명]
git checkout [원격 브랜치명]

checkout 명령어가 Git2.23버전부터 switch, restore로 분리되었습니다.
로컬 브랜치로 전환할 때는 switch , 원격 브랜치로 전환할 때는 checkout 명령어를 사용합니다.

브랜치 병합하기
me@work MINGW64 ~

git merge [브랜치명]

새로운 기능을 추가할 때 다른 브랜치에서 작업하다가 main 브랜치에서 다른 브랜치를 병합해줍니다.

충돌 해결하기

메인 브랜치와 다른 브랜치에서 동일한 파일이 서로 다르게 수정되었을 경우 충돌이 일어납니다.
충돌이 일어난 부분을 수동으로 수정한 후 커밋을 생성해주면 됩니다.
