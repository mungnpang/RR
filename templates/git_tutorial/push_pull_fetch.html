원격 저장소와 로컬 저장소 연결하기
me@work MINGW64 ~

git remote add [원격 저장소명] [원격 저장소 주소]

원격 저장소와 연결 확인하기
me@work MINGW64 ~

git remote -v

로컬 저장소의 내용을 원격 저장소로 올리기(push 하기)

me@work MINGW64 ~

git push [원격 저장소명] [브랜치명]
ex> git push origin main

원격 저장소의 내용을 로컬 저장소에 가져오기만 하기
me@work MINGW64 ~

git fetch [원격 저장소명] [브랜치명]
ex> git fetch origin main

git fetch를 입력하고 git checkout origin/main 입력해서 origin/main 브랜치를 확인해봅니다.
원격 저장소의 내용을 가져온 걸 확인할 수 있습니다.

원격 저장소의 내용을 로컬 저장소로 가져와 나의 Working Directory와 합치기
me@work MINGW64 ~

git pull [원격 저장소명] [브랜치명]
ex> git pull origin main

원격 저장소 복제하기

me@work MINGW64 ~

git clone [원격 저장소 주소]

원격 저장소가 통째로 복제됩니다.
