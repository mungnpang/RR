Git 시작하기

me@work MINGW64 ~

git config --global user.name "본인이름"
git config --global user.email "본인 이메일"

Git bash에서 진행합니다.
커밋을 생성할 때 누가 만든 것인지 명시하기 위해 설정합니다. 처음에만 설정해주면 됩니다.

me@work MINGW64 ~

git config --list

config 설정 확인하기

me@work MINGW64 ~

git config --global init.defaultBranch main

기본 브랜치명을 main으로 변경합니다.
기본 브랜치명은 master로 되어있는데 이 단어는 master/slave 를 연상시켜 권장되지 않고 있습니다.

git 저장소를 만들 프로젝트의 터미널에서 실행

me@work MINGW64 ~

git init

프로젝트 내에 git 저장소를 만듭니다. .git파일이 생성됩니다. (숨긴)
